# 🌿 Crop Disease Detection System
### Decision Tree-Based Plant Disease Classifier | Minor Project | IILM University, Greater Noida | 2025–26

---

## 📌 Project Overview

India's agricultural sector loses billions of rupees annually due to undetected plant diseases. This project develops a **software-only Crop Disease Detection System** using the **Decision Tree classification algorithm** that:

- Accepts a leaf image as input
- Extracts **10-dimensional GLCM + colour features** from it
- Classifies the leaf into one of **5 disease categories**
- Explains *why* the prediction was made using human-readable IF-THEN rules
- Delivers results through a **Flask + Bootstrap 5 web application**

## ⚙️ Setup & Installation

### 1. Clone the repository
```bash
git clone https://github.com/YOUR_USERNAME/crop-disease-detection.git
cd crop-disease-detection
```

### 2. Create a virtual environment
```bash
python -m venv venv
source venv/bin/activate        # On Windows: venv\Scripts\activate
```

### 3. Install dependencies
```bash
pip install -r requirements.txt
```

### 4. Download the dataset
- Go to: https://www.kaggle.com/datasets/mohamedhanyyy/diamos-plant-disease
- Download and extract into `data/images/`
- Structure should be:
```
data/images/
    Healthy/
    Leaf_Spot/
    Leaf_Curl/
    Slug_Damage/
    Fruit_Disease/
```

---

## 🚀 How to Run

### Step 1 — Run preprocessing & feature extraction
```bash
python src/preprocess.py
python src/feature_extraction.py
```
This creates `data/features.csv`

### Step 2 — Train the model
```bash
python src/train.py
```
This saves `models/decision_tree_model.pkl`

### Step 3 — Evaluate the model
```bash
python src/evaluate.py
```
This saves all output plots in `outputs/`

### Step 4 — Launch the web app
```bash
python app/app.py
```
Open your browser at: **http://127.0.0.1:5000**
